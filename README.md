# 微信社群自动化运营 Agent 系统

基于 Claude 大模型构建的微信社群智能运营助手，实现社群内容自动生成、用户意图识别与分层回复。

## 项目背景

社群运营耗时耗力，本项目通过 Agent 系统将每日重复性运营工作自动化，日均节省 1-2 小时人工时间。

## 核心功能

- 自动生成每日社群推文/话题内容
- 识别用户提问意图并分类回复
- 社群活跃度维护与冷启动话术
- 多社群内容差异化管理

## 技术栈

- 底层模型：Claude 系列（claude-sonnet）
- 工具：OpenClaw
- 交付形式：System Prompt + 多轮对话流程设计
